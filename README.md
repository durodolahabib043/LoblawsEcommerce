## App Architecture
This app uses the MVVM architecture
1. The View - consist of the HomeController, ProductDetailController, CartViewController and ProductCell 
2. ModelView - consist of the HomeModelView
3. Model - consist of LoblawsProduct

## App Details
1. This is built programmatically NO STORYBOARD 

## This task used the below apis 
Protocols , Delegates, Extensions, Custom Views and XCTestCase

## Unit Test
1. Unit test is added to the project , use (Command U) to run the test

  1. Xcode Version 12.3 , Swift 5
